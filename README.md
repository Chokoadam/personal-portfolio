# Adam Vasquez Portfolio

A modern personal portfolio built with React, TypeScript, Vite, Framer Motion, and regular CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Structure

```txt
src/
  components/
    About.tsx
    Contact.tsx
    ExperienceTimeline.tsx
    Footer.tsx
    Hero.tsx
    Hobbies.tsx
    Navbar.tsx
    Notes.tsx
    ProjectCard.tsx
    Projects.tsx
    Section.tsx
    SkillTag.tsx
    Skills.tsx
  data/
    portfolio.ts
  styles/
    global.css
```

Most portfolio content lives in `src/data/portfolio.ts`, including links, project cards, experience entries, skill groups, notes, and hobbies.
